This privacy policy applies to the public documentation site, any hosted BankFind MCP Server instance operated from this repository, and the related ChatGPT app listing.

Effective date: 2026-03-30

## Independent Project Notice

This is an unofficial personal project. It is not affiliated with, endorsed by, supported by, or operated on behalf of the FDIC or any other government agency, and nothing in this project should be read as a statement from the FDIC.

## What This App Does

The app helps users query public FDIC BankFind data through a natural-language interface. It does not require an account from this project and it is designed to work with public bank data rather than user-provided financial records.

## Information We May Receive

- prompts, tool inputs, and other requests you send through a hosted instance of the service
- basic technical data needed to operate the service, such as timestamps, IP address, user agent, and error logs
- messages you send through GitHub issues or other support channels

## How Information Is Used

We use this information to:

- return the requested results
- monitor reliability, investigate bugs, and prevent abuse
- improve documentation, prompts, and service behavior

## Public And Third-Party Data Sources

The app relies on public FDIC data. Requests may also pass through third-party platforms you choose to use, such as ChatGPT, GitHub, hosting providers, and the FDIC API itself. Those services have their own policies and practices.

## Data Sharing

We do not sell personal information. Information may be shared only as needed to operate the service, comply with legal obligations, protect the service from abuse, or work with infrastructure providers that help host or monitor the app.

## Data Retention

Operational logs and support messages are kept only for as long as they are reasonably needed for security, debugging, and maintenance.

## Self-Hosted Deployments

If you run this MCP server yourself, you are the operator of that deployment and this policy does not control the data practices of your environment.

## Security

Reasonable steps are taken to protect service data, but no internet service can guarantee absolute security.

## Changes

This policy may be updated as the project changes. The effective date above reflects the latest published version.

## Contact

For privacy questions about this project, open an issue at <https://github.com/jflamb/fdic-mcp-server/issues>. For security-sensitive reports, follow the guidance on the [Security]({{ '/security/' | relative_url }}) page.
