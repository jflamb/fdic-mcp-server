## Highlights

This patch release improves reliability for longer-running usage, adds clearer warnings when comparisons are incomplete, and broadens coverage across the available tools.

## Improved

- Improved stability for longer-running local and hosted sessions.
- Added warnings when large comparison requests may be incomplete because the FDIC response was truncated.
- Reduced duplication in the shared analysis paths used by comparison tools.
- Expanded MCP HTTP coverage for failures, history, SOD, and annual summary tools.
- Clarified maintainer guidance in `AGENTS.md` and normalized stale completed checklist items in `tasks/todo.md`.

## Notes

- The npm package remains available as `fdic-mcp-server`.
- The hosted HTTP endpoint remains `https://bankfind.jflamb.com/mcp`.
- This release is primarily a reliability and maintenance update rather than a new-feature release.
