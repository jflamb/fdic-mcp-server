## Highlights

This patch release improves parameter discovery for the analysis tools so MCP clients can present them more clearly during setup and use.

## Fixed

- Fixed the advertised input details for `fdic_compare_bank_snapshots` and `fdic_peer_group_analysis`.
- Preserved argument validation for both tools while fixing their published input details.
- Added HTTP regression coverage for tool discovery and invalid analysis requests.

## Notes

- This release is intended to address client-facing parameter discovery issues in hosts such as Claude Desktop.
- The npm package remains available as `fdic-mcp-server`.
- The hosted HTTP endpoint remains `https://bankfind.jflamb.com/mcp`.
