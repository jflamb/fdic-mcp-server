## Highlights

This patch release fixes MCP tool schema advertising for the server-side analysis tools so MCP clients can render their parameters correctly from `tools/list`.

## Fixed

- Advertised full input schemas for `fdic_compare_bank_snapshots` and `fdic_peer_group_analysis` instead of empty parameter objects.
- Preserved cross-field argument validation for both tools after separating MCP schema listing from runtime validation.
- Added HTTP regression coverage for the listed schemas and invalid analysis-tool argument combinations.

## Notes

- This release is intended to address client-facing parameter discovery issues in hosts such as Claude Desktop.
- The npm package remains available as `fdic-mcp-server`.
- The hosted HTTP endpoint remains `https://bankfind.jflamb.com/mcp`.
