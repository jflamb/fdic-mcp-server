# v1.2.0 — CAMELS Analysis Tools

## New Tools

This release adds three new MCP tools that produce CAMELS-style analytical assessments from FDIC BankFind financial data.

### `fdic_analyze_bank_health`

Run a comprehensive health assessment for a single institution. Scores five CAMELS components — Capital Adequacy, Asset Quality, Earnings, Liquidity, and Sensitivity to Market Risk — and derives a weighted composite rating from 1 (Strong) to 5 (Unsatisfactory). Includes trend analysis across prior quarters and automatic risk signal detection.

### `fdic_compare_peer_health`

Rank a group of institutions by CAMELS-style health scores. Supports three selection modes: explicit CERT list, state-wide scan, or asset-size range. Sort by composite rating or any individual component. Useful for identifying the strongest and weakest banks in a peer group.

### `fdic_detect_risk_signals`

Screen institutions for early warning indicators. Classifies signals by severity:

- **Critical**: undercapitalized (Tier 1 leverage below 5%), operating losses (negative ROA), reserve coverage below 50%
- **Warning**: any CAMELS component rated 4 or worse, significant deteriorating trends, brokered deposits above 15%, noncurrent loans above 3%
- **Info**: moderate deteriorating trends

Results are ranked by signal severity so the most concerning institutions appear first.

## Shared Scoring Engine

All three tools share a scoring engine (`camelsScoring.ts`) containing metric definitions, regulatory-informed thresholds, weighted component scoring, and linear-regression trend analysis. The engine is implemented as pure functions with no external dependencies, making it fully testable in isolation.

## Important Disclaimer

These tools produce analytical assessments based on publicly available FDIC financial data. They are **not** official CAMELS ratings assigned by bank examiners. The Management (M) component is omitted because it cannot be assessed from public data. The Sensitivity (S) component uses proxy metrics (NIM trend and securities concentration) rather than direct interest rate risk measures.
