# v1.3.0 — Advanced Analysis Enhancements

## Overview

This release adds eight new MCP tools across five analytical tracks, significantly expanding the server's ability to support off-site examination workflows. All tools follow the existing pattern of text + structured content output and require no additional credentials for Tracks 1-4. Track 5 optionally uses the FRED API for economic context.

## New Tools

### Credit, Funding, And Securities Detail (Track 1)

#### `fdic_analyze_credit_concentration`

Analyze loan portfolio composition and credit concentration risk. Computes CRE and construction concentrations relative to capital per the 2006 interagency guidance (OCC/FRB/FDIC), breaks down the loan book by type (CRE, C&I, consumer, residential RE, agricultural), and flags concentrations exceeding regulatory thresholds.

#### `fdic_analyze_funding_profile`

Evaluate deposit composition and funding stability. Measures core deposit reliance, brokered deposit exposure, wholesale funding dependence, and FHLB advance utilization. Flags institutions with elevated wholesale funding, high brokered deposits, or low core deposit ratios.

#### `fdic_analyze_securities_portfolio`

Assess securities holdings and portfolio concentration. Reports securities as a percentage of assets and capital, MBS concentration within the portfolio, and flags high concentration levels.

### UBPR-Equivalent Ratio Analysis (Track 2)

#### `fdic_ubpr_analysis`

Compute UBPR-equivalent ratio analysis organized by standard UBPR page sections: summary ratios (ROA, ROE, NIM, efficiency), loan mix, capital adequacy, liquidity, and year-over-year growth rates. Ratios are computed directly from FDIC Call Report data and are clearly labeled as UBPR-equivalent, not official FFIEC UBPR output.

### Market Share And Franchise (Track 3)

#### `fdic_market_share_analysis`

Compute deposit market share for any MSA or county using Summary of Deposits data. Ranks institutions by deposit share, calculates the Herfindahl-Hirschman Index (HHI) with DOJ/FTC concentration classification (unconcentrated, moderately concentrated, highly concentrated), and optionally highlights a specific institution's competitive position.

#### `fdic_franchise_footprint`

Map a single institution's branch network across all markets. Groups branches by MSA, shows deposit totals and branch counts per market, and identifies where the institution's franchise value is concentrated.

### Holding Company (Track 4)

#### `fdic_holding_company_profile`

Profile a bank holding company by grouping its FDIC-insured subsidiaries and aggregating financial metrics. Look up by holding company name or by any subsidiary's CERT number. Shows consolidated assets, deposits, asset-weighted ROA and equity ratio, and a comparison table of individual subsidiary performance.

### Regional Economic Context (Track 5)

#### `fdic_regional_context`

Overlay macro/regional economic data on a bank's operating environment using FRED (Federal Reserve Economic Data). Provides state unemployment trends, state-vs-national comparison, federal funds rate environment classification, and a narrative assessment of economic conditions affecting bank performance. Gracefully degrades with a helpful message if the FRED API is unavailable.

Set the `FRED_API_KEY` environment variable for reliable FRED access at higher rate limits. The tool works without a key for low-volume use.

## Shared Engines

Each track includes a pure-function scoring engine in `src/tools/shared/` that is fully testable in isolation:

- `creditConcentration.ts` — loan-type decomposition and interagency threshold evaluation
- `fundingProfile.ts` — deposit composition and wholesale funding risk assessment
- `securitiesPortfolio.ts` — securities concentration and portfolio risk scoring
- `ubprRatios.ts` — UBPR-equivalent ratio groups and growth rate computation
- `marketShare.ts` — deposit aggregation, market share ranking, and HHI calculation
- `holdingCompany.ts` — subsidiary grouping and asset-weighted metric aggregation
- `regionalContext.ts` — macro trend classification and narrative generation

## New Service

- `fredClient.ts` — FRED API client with URL construction, response parsing, and state-to-series mapping
